HTML is a Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. HTML elements are the building blocks of HTML pages. HTML elements are represented by tags. HTML tags label pieces of content such as "heading", "paragraph", "table", and so on. Browsers do not display the HTML tags, but use them to render the content of the page

Each page contains:


HTML elements

What consitutes a valid HTML page?

All HTML pages contain the following elements which determine the functionality of the page in chronological order:

  1. <!DOCTYPE html> :

    must be the very first thing in your HTML document, before any other tag. The <!DOCTYPE> declaration is not an HTML tag; it is an instruction to the web browser about what version of HTML the page is written in.
    w3schools Link

  2. <html> :

    tells the browser that this is an HTML document. Required in all HTML documents.

    <html>body element</html>

    w3schools Link

  3. <head> :

    This element is a container for metadata (data about data) and is placed between the html tag and the body tag.

    <head>
    head element
    </head>

    w3schools Link

  4. <body> :

    contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc...

    <body>
    body element
    </body>

    w3schools Link

  5. <title> :

    required in all HTML documents. Defines the title of the document

    <title>
    title content</title>

Lists

Description:

There are 3 ways to show lists

Unordered lists: It starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles) by default (but can be changed):

Syntax:

<ul>
<li>list here</li>
</ul>

Example:

  • Coffee
  • Tea
  • Milk

Oredered Lists: It starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers by default (but can bec changed through the type attribute)

Syntax:

<ol>
<li>list here</li>
</ol>

Example:

  1. Coffee
  2. Tea
  3. Milk

A Description List: A description list is a list of terms, with a description of each term. The <dl> tag defines the description list, while the <dt> tag defines the term (name), and the <dd> tag describes each term

Example:

Coffee
- black hot drink
Milk
- white cold drink

Tables

Description:

An HTML table is defined with the <table> tag.
Each table row is defined with the <tr> tag.
A table header is defined with the <th> tag.
A table data/cell is defined with the <td> tag.

Syntax:


<table>
<tr>
<th>heading of columns</th>
<td>info of table</td>
</tr>
</table>

Examples:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
